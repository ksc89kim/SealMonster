SealMonster 설명
-------------
1. 몬스터를 수집하고, 해당 몬스터와 싸움을 하는 게임.
2. 스택 배틀 게임이 있음. (스킬을 쌓을 수 있고 쌓여진 스킬로 콤보및 강화 할 수 있음.)
3. 씰몬스터에 등급이 있음. (L,SSS,SS,S,A,B,C,D,E,F,G)
4. 씰몬스터는 대략 3000여종이 있음.
5. 퀘스트가 있음. 
6. 속성이 있음.
